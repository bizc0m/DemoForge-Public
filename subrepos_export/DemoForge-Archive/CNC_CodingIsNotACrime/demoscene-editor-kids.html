Mini Demo Maker

Un mini editeur demoscene HTML tres simple. Clique sur des gros boutons, ecris ton texte, et regarde la magie.

5. Actions

Sources d'inspiration

Technique generale: CODEF, Canvas Oldschool Demo Effects Framework pour l'idee de faire des demos oldschool dans le navigateur avec HTML5 Canvas.
Tutoriels debutants: Mellow Man's guide to CODEF, notamment l'approche pas-a-pas, le canvas, les gradients, les rasters, le starfield et le scrolltext.
Culture demoscene: Democyclopedie - Differents types de demos pour les notions de menu, megademos, intros et scrollers.
Liste des effets: Democyclopedie - Liste des effets pour les effets retenus ici: plasma, rasters, menu et scrollers.
Liens Perplexity donnes par l'utilisateur: gardes comme pistes editoriales, mais leur contenu n'etait pas lisible directement depuis mon environnement. Les choix ci-dessus sont donc bases sur les sources publiques accessibles et sur une inference prudente a partir des titres.

Cette version reste volontairement hyper simple: pas besoin de coder. Tu cliques, tu ecris, tu joues.
