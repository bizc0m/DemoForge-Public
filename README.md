# DemoForge - GitHub Ready Chrono

## Ouvrir

Ouvrir `index.html` a la racine.

## Structure

- `index.html` : hub visuel avec miniatures, ordre antichronologique.
- `interfaces/chrono/` : HTML renumerotes en `1.1`, `1.2`, etc.
- `assets/thumbs/` : screenshots en petit pour reperage visuel.
- `interfaces/`, `releases/`, `docs/`, `audio/`, `code-data/` : elements originaux au format GitHub.
- `manifest.json` : correspondance numero chrono -> source originale -> screenshot.
