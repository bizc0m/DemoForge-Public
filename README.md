# DemoForge-Public

DemoForge is my first public release: a small project that started as a random idea and slowly turned into a tiny DemoMaker.

Please be kind — I’m still learning, and I honestly understand very little about GitHub :P

> Started as a hidden screen experiment...
> 3 weeks later... a little DemoMaker appeared.

## GitHub-ready archive

This repository has been rebuilt as a visual archive for browsing DemoForge interfaces.

- `index.html`: visual hub with compact thumbnails in reverse chronological order.
- `interfaces/chrono/`: HTML interfaces renumbered as `1.1`, `1.2`, etc.
- `assets/thumbs/`: visual screenshots for quick orientation.
- `interfaces/`, `releases/`, `docs/`, `audio/`, `code-data/`: original project elements.
- `manifest.json`: mapping from chrono number to original source and screenshot.

## Original idea

The first goal was to create a hidden screen for a future project.
It reminded me of DCK on Atari, which I absolutely loved.

Then I thought:

> What if this became a fun and educational tool for kids — a way to make demos and learn about demoscene culture?

The concept is simple:

- Code
- Effect layer
- Description
- Author
