About the project
Hi! This project started as a random fun idea… and somehow turned into this.

I’m not a dev. Not technical. Like… not at all.
(And yes, I really don’t understand GitHub 😅)

But hey — the “code” is here:
https://github.com/bizc0m

Help is very welcome if you feel like it 🙏

The original goal was to build a hidden screen for an upcoming project.
It reminded me of DCK on Atari, which I absolutely loved.

Then I thought:
“What if we made something fun and educational — a tool for kids to create demos and discover demoscene culture?”

The idea:
“Code” + effect layer + description + author
(It used to work better… we’ll get there again 😅)
