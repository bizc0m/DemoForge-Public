## The original idea

The first goal was to create a hidden screen for a future project.  
It reminded me of DCK on Atari, which I absolutely loved.

Then I thought:

> What if this became a fun and educational tool for kids — a way to make demos and learn about demoscene culture?

The concept is simple:

- Code
- Effect layer
- Description
- Author

It used to work better than it does now, but I’m still improving it.

## How it is built

This is pure vibe coding.  
We work with what we can, and sometimes things break, disappear, or come back in weird ways.

Most of the complex parts are done with:

- Codex / Emergent.sh when needed
- Claude

I’d genuinely love for this to become a collaborative project.  
That’s also why I introduced a modular structure.

And if possible... let’s keep it within **1.44 MB** 😉
